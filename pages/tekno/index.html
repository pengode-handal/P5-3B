Teknologi

Dengan perkembangan teknologi dan globalisasi yang gencar terjadi, kearifan lokal mulai memudar. Maka dari itu, diperlukan adanya suatu inovasi agar kearifan lokal tetap terlestarikan. Daripada harus bersaing dengan perkembangan teknologi, bagaimana jika kearifan lokal dipadukan saja dengan teknologi? Hal tersebutlah yang melatarbelakangi kegiatan yang kami lakukan, Divisi Teknologi, membuat berbagai projek teknologi dengan tujuan untuk melestarikan kearifan lokal dan memperkenalkannya ke masyarakat luas melalui website, game, poster, serta foto dan video dengan tema kearifan lokal.

Divisi Teknologi

Web Design

Website Gelar Karya
GitHub

Divisi Web Design membuat website sebagai sarana informasi mengenai Gelar Karya "Kearifan Lokal". Kami beranggapan bahwa website ini adalah wadah yang tepat untuk memberikan informasi terutama pada era digitalilasi ini. Kami berharap projek pembuatan website ini dapat menginspirasi seseorang untuk menyebarkan kearifan lokal dengan bantuan teknologi.

Game Developer

Membuat game roblox mengenai kearifan lokal
Roblox

Bidang Teknologi membuat divisi game development untuk gelar karya kali ini. Tugas dari divisi ini adalah untuk membuat game mengenai kearifan lokal. Kami memilih platform Roblox sebagai engine dalam pembuatan game kami. Latar dari game ini terinspirasi dari Keraton Yogyakarta dikarenakan Keraton Yogyakarta adalah salah satu ikon kearifan lokal di Provinsi Daerah Yogyakarta. Di game ini akan ada quiz atau pertanyaan tentang kearifan lokal dan pemain diharapkan untuk menjawab pertanyaan dengan benar
